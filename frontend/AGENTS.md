# Frontend Agent 指南

## 技术栈

- Vue 3。
- Vite。
- 图标使用 `lucide-vue-next`。

## 目录与实现约定

- 当前前端主要入口在 `src/App.vue`，样式在 `src/style.css`。
- 接口调用统一走 Vite `/api` 代理，不在组件里写死后端域名。
- 用户可见文案使用中文。
- 上传 Markdown 入口必须保留前端扩展名校验，后端做最终校验。
- AI 分析区域只展示任务状态、失败原因和 scene 草稿，不在当前阶段做 YAML 下载和 scene 在线编辑。

## 交互与样式约束

- 操作按钮优先搭配 lucide 图标。
- 错误信息既要写入页面状态，也可以在关键异步流程失败时给出明确提示。
- 不新增营销式落地页；页面应保持工具型、操作型体验。
- 修改 UI 后检查移动端布局不溢出、不遮挡。

## 验证

- 前端改动至少运行：`npm run build`。
- 涉及接口流程时，配合后端启动后通过 `/api` 代理验证。
